## 🎂 Birthday Animation

An interactive envelope animation that reveals a birthday surprise!

### How to View It

**Option 1: GitHub Pages (Recommended)**
1. Visit: `https://cubepro123.github.io/learning-to-code/birthday-animation.html`
   - *Note: GitHub Pages needs to be enabled in repository settings first*

**Option 2: View Locally**
1. Clone or download this repository
2. Open `birthday-animation.html` directly in your web browser
3. Click on the envelope to see the surprise!

**Option 3: HTMLPreview (Quick View)**
Visit: [View on HTMLPreview](https://htmlpreview.github.io/?https://github.com/cubepro123/learning-to-code/blob/main/birthday-animation.html)

### How It Works
- Click the envelope to open it
- Watch the flap rotate and letter lift up
- After 2 seconds, a birthday surprise appears with balloons and a dancing cake! 🎈🎂

## Setting Up GitHub Pages (One-time setup)

To enable the GitHub Pages link above:
1. Go to repository **Settings**
2. Scroll to **Pages** section (left sidebar)
3. Under **Source**, select: `Deploy from a branch`
4. Choose branch: `main` and folder: `/ (root)`
5. Click **Save**
6. Wait a few minutes for deployment
7. Your animation will be live at: `https://cubepro123.github.io/learning-to-code/birthday-animation.html`
